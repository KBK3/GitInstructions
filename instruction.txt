Необходимые комманды git
1. Создание локального репозитория.
	git init
2. Проверка состояния репозитория.
	git status
	=====Работа с файлами=====
3. Добавление файлов в репозиторий.
	git add <fileName> or .
4. Удаление ошибочно добавленных в git файлов.
	git reset HEAD <fileName>
5. Создание файла .gitignore .
	touch .gitignore
6. Запись в .gitignore игнорируемых файлов и дерикторий.
	echo "FileName" >> .gitignore,
	echo "FolderName" >> .gitignore
	=====Работа с файлами=====
7. Добавление изменений в репозиторий.
	git add <fileName> or .
8. Создание слепка состояния файлов.
	(Перед этим проверить чтобы файлы были добавленны в git.Пункт.3.) 
	git commit -m "Подробное описание работ, которые были совершены"
9. Создание репозиторийя на github.com c именем репозиторийя локального.
10. Связывание удаленного и локального репозитория.
	git remote add origin master (ссылка на репозиторий на github.com)
11. Проверка с каким удаленным репозиторием связан локальный.
	git remote -v
12. Как удалить связь с удаленным репозиторием.
	git remote rm origin master
13. Отправка файлов из локального репозитория на удаленный.
	git push -u origin master 
14. Как вытянуть изменения из удаленного репозитория.
	git pull <Ссылка на репозиторий>
